# 🦁 Kenya Tourism RAG System

Retrieval-Augmented Generation (RAG) system for Kenya travel and tourism information. Uses local embeddings (HuggingFace) and Grok (xAI) for grounded, hallucination-free responses.

## 🚀 Quick Start

### 1. Installation

```bash
# Clone repository
git clone &lt;your-repo-url&gt;
cd kenya_tourism_rag

# Create virtual environment
python -m venv venv

# Activate (macOS/Linux)
source venv/bin/activate

# Activate (Windows)
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
